## 📂 Fichiers du Projet

### Fichiers de Code
```
app/
├── layout.tsx          → Layout principal de l'application
├── page.tsx            → Page d'accueil avec toutes les sections
├── not-found.tsx       → Page 404 personnalisée
└── globals.css         → Styles globaux et variables CSS

components/
├── header.tsx          → Navigation avec smooth scroll
├── hero.tsx            → Section héro avec animations
├── story.tsx           → Section "Notre Histoire"
├── mission.tsx         → Section "Notre Mission"
├── how-it-works.tsx    → Section "Comment ça marche"
├── team.tsx            → Section "Notre Équipe"
├── impact.tsx          → Section "Notre Impact"
├── contact.tsx         → Formulaire de contact
└── footer.tsx          → Pied de page

hooks/
└── use-smooth-scroll.ts → Hook personnalisé pour le scroll fluide

public/
├── .htaccess           → Configuration Apache pour OVH
└── 500.html            → Page d'erreur serveur
```

### Fichiers de Configuration
```
next.config.mjs         → Configuration Next.js (export statique)
package.json            → Dépendances et scripts
tsconfig.json           → Configuration TypeScript
tailwind.config.ts      → Configuration Tailwind CSS
postcss.config.mjs      → Configuration PostCSS
```

### Scripts et Outils
```
deploy.ps1              → Script PowerShell de build automatisé
```
